# Code of conduct

This project uses a concise conduct standard for research collaboration.

## Expected behavior

- Be precise, evidence-oriented, and respectful.
- Critique claims and methods directly without personal attacks.
- Distinguish supported results from speculation.
- Preserve attribution for prior work and contributor evidence.

## Unacceptable behavior

- Harassment, threats, or discriminatory conduct.
- Publishing secrets, private credentials, private filing materials, or private correspondence.
- Knowingly overstating results or removing limitation language from evidence records.
- Misrepresenting QRoPE as proving broad quantum advantage or transformer superiority without supporting evidence.

## Enforcement

Maintainers may edit, close, or remove issues, discussions, pull requests, or comments that violate this standard. Serious issues should be reported privately using the contacts in `SECURITY.md`.

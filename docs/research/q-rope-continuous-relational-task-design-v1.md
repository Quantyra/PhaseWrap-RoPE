# Research note

## Task direction
- Preserve one next task family only:
  - `continuous_relational_target`

## Core idea
- Stop using binary labels derived from a tiny set of agreement booleans.
- Use a continuous target defined over relational structure, such as a graded contrast or coupled response magnitude.
- The next task should require the model to predict a value, not just classify an exact parity cell.

## Why this is the right next memo
- Every discrete agreement-family task so far has eventually been matched by an explicit symbolic control once the right interaction term was exposed.
- A continuous relational target raises the bar by removing the small finite truth table that bounded symbolic controls can memorize directly.

## Protocol boundary
- Memo-only.
- No implementation approval in this step.
- No second alternative family in parallel.

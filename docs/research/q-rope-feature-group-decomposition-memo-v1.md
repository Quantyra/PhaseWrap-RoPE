# Q-RoPE Feature Group Decomposition Memo v1

Date: 2026-03-11
Stories: S903

## BLUF
The standing witness advantage is best interpreted as a latent-conditioned relational response advantage built from a small number of structured interaction groups, not from broad uncontrolled complexity.

## Declared Summary Groups
The frozen symbolic families were allowed to use only declared summary structure built from:
- coarse transition indicators
- first-order analog summaries
- bounded pairwise cross-direction summaries
- bounded chart/atlas families with frozen global partitions
- bounded higher-order polynomial expansions within the declared family contract

## Witness-Only Latent Groups
The witness families add latent-conditioned response structure that the symbolic families intentionally exclude.

### Benchmark Witness
On the standing benchmark, the witness carries the gap through:
- latent transition phase response
- latent transition curvature response
- latent-conditioned mixing between declared analog summaries and transition-local latent structure

### Path Transfer Witness
On the bounded path transfer family, the witness composes those transition responses into:
- path phase mean
- path phase gap
- path curvature mean
- path curvature product
- path latent-declared mixing
- path latent cross-curvature

### Loop Transfer Witness
On the bounded loop transfer family, the witness preserves the same basic mechanism class while changing the relational topology from path-local to loop-closure interactions.

## Working Decomposition
The most defensible internal decomposition is:
1. declared analog summaries provide the coarse signal floor
2. latent-conditioned transition response creates structured residual signal
3. path/loop composition determines whether that residual signal transfers across relational families

## What This Rejects
This decomposition rejects several weaker explanations:
- simple linear declared-summary advantage
- small bounded polynomial advantage over declared summaries alone
- frozen atlas partition advantage alone
- minor nuisance sensitivity as the main driver of the result

## Limits
- This is still a memo-level decomposition, not a formal theorem.
- The memo does not claim uniqueness against all possible symbolic families.
- The memo does not claim hardware relevance.

# Orthogonalized continuous task design

## Motivation
The state-sensitive continuous task still failed because coarse agreement structure remained predictive enough for a simple symbolic control.

## Preserved next angle
- future task family: `orthogonalized continuous relational response`

## Requirement
A future target should be constructed so that coarse agreement terms have near-zero standalone predictive value, for example by:
- centering target contributions within each coarse agreement tuple
- making the target depend primarily on within-state analog contrasts
- auditing that a coarse lookup regressor cannot exceed a low baseline before any witness implementation is approved

## Future restart bar
Do not approve a future implementation unless the task memo proves that coarse-state lookup is intentionally de-powered rather than merely supplemented.

# Q-RoPE Program Status and Decision Memo v1

## Program status
Overall status: `viable but claim-bounded`

The program has succeeded in establishing:
- a coherent novelty direction
- a reproducible evaluation protocol
- real remote execution feasibility on both a photonic provider path and an IBM comparator path

The program has not established:
- stable remote superiority
- stable cross-provider agreement
- a robust local redesign path that clearly improves on the current primary reference

## What is defensible now
1. Novelty framing
- Q-RoPE remains a coherent relative-phase positional-encoding proposal for hybrid quantum attention.

2. Protocol quality
- the repo contains strong process discipline, reproducibility scaffolding, evidence logging, and git checkpoint history.

3. Remote feasibility
- the program has real remote execution evidence on direct Quandela and IBM Runtime.

4. Empirical boundary clarity
- the current remote packet shows strong seed sensitivity and provider dependence.
- the current local redesign line has not produced a robust improvement beyond the parity screening upgrade.

## What is not defensible now
1. stable remote performance advantage claims
2. hardware-style triumph claims
3. provider agreement claims
4. any statement that implies the local redesign branch has found a strong successor to `V3`

## Publication posture
Best current posture:
- bounded method-plus-evidence paper

Recommended emphasis:
- novelty of the relative-phase framing
- reproducible comparator stack
- remote feasibility with explicit instability
- disciplined negative/neutral interpretation where appropriate

Do not emphasize:
- a universal gain story
- “wins on hardware”
- stable cross-provider ranking

## Bottom line
Q-RoPE still looks like a legitimate research initiative.
The strongest current story is:
- novel method framing
- real remote feasibility
- disciplined empirical boundaries

The weakest current story is:
- local mechanism improvement beyond the baseline path

So the correct near-term move is not more technical thrashing.
It is synthesis, positioning, and claim discipline.

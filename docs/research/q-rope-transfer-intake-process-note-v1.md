# Q-RoPE Transfer Intake Process Note v1

Date: 2026-03-12
Stories: S988

## BLUF
Future transfer execution now requires both a completed intake memo and a completed review checklist before any implementation-approval gate is drafted.

## Process
1. Fill the transfer candidate intake template.
2. Score the candidate against the review checklist.
3. If the checklist result is `PASS`, draft one implementation-approval gate.
4. If the checklist result is `FAIL` or `HOLD`, keep the line memo-only.

## Reason
This prevents the repo from drifting back into low-value branch churn and makes the survivor-vs-braid screen operational instead of informal.

# Transition Orbit Rank-Stable Symbolic Lookup Proof v1

## Prohibited shortcut family
- any bounded symbolic ordinal lookup that maps a small coarse orbit-transition state directly to a rank bucket or monotone surrogate

## Required proof on paper
- the future task must vary ordinal target values within the same coarse orbit-transition state
- the future task must require ordered transition structure plus within-state analog variation
- the future task must declare at least one bounded symbolic ordinal control that is expected to fail if the task is well formed

## Failure condition
- if one small symbolic ordinal lookup can explain the target without ordered analog interaction, the line should not be implementation-approved
